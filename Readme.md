## How to use
install package
```
pub get
``` 

run the app
```
dart bin/main.dart
```

compile
```
dart2aot bin/main.dart bin/main.dart.aot
```

run compiled app
```
dartaotruntime bin/main.dart.aot

// Speed measurment
time dartaotruntime bin/main.dart.aot
```
